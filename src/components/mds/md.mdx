# Paper Overview

### Method Compare
![](./images/method-comparison.png)

Comparison of different prompt compression paradigms for long-context LLMs. BEAVER targets a higher abstraction level than token pruning by selecting coherent pages with training-free semantic and lexical signals.

### Overall Pipeline
![](./images/beaver-pipeline.png)

- **Segmenter** maps variable-length text into structured 2D page tensors using natural delimiters, optimizing hardware efficiency while preserving local context boundaries.
- **PageEncoder** reuses the target model's embedding table with dual-path pooling to capture global semantics and salient local cues.
- **QueryPlanner** selects valuable pages with hybrid semantic-lexical scoring and structural priors: Anchor pages for global metadata, Flow pages for local coherence, and Flash pages for distant critical evidence.

![](./images/sentence-smoothing.png)

Sentence smoothing extends selected page fragments outward to the nearest sentence boundaries, repairing truncated spans and preserving syntactic coherence in the final compressed context.

### Experiment Details

We evaluated BEAVER on LongBench, ZeroSCROLLS, RULER, and L-Eval with fixed hyperparameters across benchmarks. LongBench and ZeroSCROLLS use 2,000 and 3,000 token budgets, RULER uses a 3,000 token budget at 16k context length, and L-Eval uses a 2,000 token budget for out-of-domain generalization. Experiments compare against unsupervised statistical methods, supervised compression methods, embedding retrieval baselines, and recent training-free methods.

### Results

BEAVER establishes the strongest training-free results on LongBench and ZeroSCROLLS, sets new training-free state of the art on RULER and L-Eval, and scales to one million tokens with 10.4 seconds of compression latency. At the 1M-token scale, it is 26.9x faster than LongLLMLingua while avoiding the Transformer-block overhead of attention-heavy compressors.

### Qualitative Comparison

The updated qualitative cases stress strict token budgets across finance QA, government-report summarization, few-shot mathematical reasoning, and code understanding. BEAVER preserves price rules, procedural exceptions, chain-of-thought examples, and code syntax more reliably than token-level compression baselines.

### Conclusion

We introduce BEAVER, a training-free prompt compression framework that recasts long-context compression as structured operations over a segment-page tensor. It preserves discourse integrity through page-level selection and sentence smoothing, delivers strong retrieval and QA performance across benchmarks, and provides practical high-throughput compression at million-token scale without task-specific training.
